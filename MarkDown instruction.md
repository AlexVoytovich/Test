## Списки

Чтобы добавить ненумерованные списки необходимо пункты выделить звездочкой (*), например так:
* Элемент один
* Элемент два
* Элемент три

Чтобы добавить нумерованные списки необходимо пункты просто пронумеровать, например так:
1. первый пункт
2. второй пункт
